# INT Reference Pipeline

This directory contains a snapshot of the INT photometry pipeline
used as a reference / benchmarking tool during development of the
QUB photometry pipeline.

This code is **not maintained** here and is **not the primary pipeline**.
It is included for comparison, reproducibility, and methodological context.
